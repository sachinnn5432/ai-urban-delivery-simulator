# AI Urban Delivery Simulator 🚗🌆

A Python-based AI Urban Delivery Simulator implementing multiple AI search and optimization algorithms for simulating smart city delivery routing.

## 📌 Project Overview

This project simulates an AI-powered delivery system in an urban environment using classic AI algorithms including:

- **BFS** (Breadth-First Search)
- **DFS** (Depth-First Search)
- **A\*** (A-Star Search)
- **Hill Climbing**
- **Minimax**
- **TSP** (Travelling Salesman Problem)

## 🛠️ Tech Stack

- **Language**: Python 3.x
- **3D Visualization**: VPython
- **Report Generation**: ReportLab

## 📁 Project Structure

```
minor project ai/
```

## 🚀 How to Run

1. Install dependencies:
   ```bash
   pip install vpython reportlab
   ```

2. Run the simulator:
   ```bash
   python main.py
   ```

3. Generate the PDF report:
   ```bash
   python generate_report.py
   ```
